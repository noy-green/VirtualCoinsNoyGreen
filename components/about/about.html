About The Project

This project is written in the following languages: javascript, html and css. The model that is built according to it is a "single page" model On the home page you can see all the latest virtual coins, by clicking on the "More information" button you can see their value in currencies in euros, dollars and shekels. You can select up to five coins and go to the "Real Time Reports" page, on this page you can see the value of the coins in dollars at any given moment.
